# Officer invite types

Alliance HQ offers three ways to bring people onto your alliance. Use this guide when you are not sure which channel fits.

## Quick comparison

| Type | Best for | Where to share |
|------|----------|----------------|
| **Invite link** | Leadership (officer, data entry, viewer) | DMs only |
| **Join code** | Broad onboarding (mailers, pinned Discord posts) | Public channels OK |
| **Commander claim** | A specific roster Commander you already know | DMs only |

Open **Settings → Team** and use the invite wizard to create any of these.

## Invite links

Use invite links to get **leadership** onto HQ.

- Share **individually in DMs**
- **Do not** share in group chats or public forums
- Protected links include a passphrase (send link and passphrase in separate messages)
- Email-bound links only work for the address you specify — HQ does not send the email for you

## Join codes

Use join codes for **broad** outreach: alliance mailers, pinned Discord posts, and similar.

- **Officer** join code with **10 uses** — usually enough for leadership
- **Member** join code with **90 uses** — usually enough for all R1s, R2s, and R3s
- Set max uses high enough for everyone who needs one

Join codes are multi-use. Recipients sign in to HQ and redeem the code.

## Commander claim

Use commander claim when you know **exactly who** should own a specific Commander on your roster.

- Share in **in-game DM** (recommended), Discord DM, or another private message
- **Do not** share in group or public forums
- Each code is **single-use** and tied to one Commander

## Ashed seat invites (automatic)

When your alliance **first connects Ashed**, HQ emails remaining Ashed seat-holders (collaborators) with an invite to join. Officers do not create these manually.
