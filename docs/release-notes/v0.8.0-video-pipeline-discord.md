## Summary

- **Video upload survey** — optional pre-upload questions (row count, scroll style) and an in-form local video player.
- **Richer pipeline diagnostics** — job and frame timing, skipped frames, and OCR overlap metrics for maintainers.
- **Admin gallery** — carousel and slideshow mode for reviewing extracted frames; zoom and momentum swipe in Job Studio.
- **Quality scoring** — automatic quality buckets on submit (`perfect` through `dropped_the_ball`); filterable in the admin video jobs list.
- **Shadow pass comparison** — linked upload groups run a primary and shadow parse pass; reviewers pick the best result and can accuracy-vote; admin shows selected pass, recommendation, and reason badges.
- **Review and post-submit UX** — split review layout with source video; after submit, an event scores page for updates without re-entering review; OCR thumbs-up/down with reason buckets.
- **Submit fixes** — Alliance Exercise event linkage, Zombie Siege payload shape, and auto-provision Ashed events when none exist yet.
- **Multi-tenant Discord VR bot** — per-alliance Ashed credentials, `/link-alliance` guild registration, up to five in-game characters per Discord user, context-aware `/help`, and Portuguese bot strings.
