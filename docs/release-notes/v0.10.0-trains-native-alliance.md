## Summary

- **Train Conductor ritual (Phase 0 + 1)** — alliance officers plan the week, roll conductor and VIP picks (slot-machine wheel, pools, VS/event auto-rolls), lock the day, and review conductor history. Game weeks follow UTC−2 server time with VS push and economy pivot templates.
- **Per-alliance game season** — season key resolves from owner override, Monday cpt-hedge sync, or server-age fallback; editable in Settings for alliance admins.
- **Native alliance mode** — alliances can run on HQ without an Ashed bearer token; roster OCR import seeds `alliance_members` for train pools and member matching.
- **HQ access invites** — owners can invite collaborators via tokenized links; invite accept flow lands in the connect path.
