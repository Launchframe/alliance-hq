## Working notes

- **PR #364 — Alliance video jobs console:** Designated processors get `/tools/video-jobs` with inspect, requeue, reprocess, and analytics, scoped to the session alliance via `/api/tools/video-jobs/*`.
- **PR #361 — Train channel setter:** R4+ can set Discord train channels by default; owners can restrict `/set-train-channel` to R5-only (`0113_train_channel_setter_min_rank`). Split PATCH RBAC so owners can update min rank without `trains:write`.
- **PR #365 — Discord VR chart PNGs:** Embed Inter for legible chart axis labels; scope no-args `/what-is-my-vr-chart` to the caller’s commander; optional `with`/`with-2`/`with-3`/`with-4` overlays plus a commander legend on every Discord PNG.

## Summary

- **Processor video jobs** — Designated video processors get an alliance-scoped jobs console at `/tools/video-jobs` to inspect, requeue, reprocess, and view analytics without the platform admin portal.
- **Discord train channels** — R4+ officers can run `/set-train-channel` by default; alliance owners can lock it back to R5-only in train Discord settings.
- **Discord VR charts** — Clearer PNG chart text, caller-scoped default charts, and optional commander overlays with a legend on `/what-is-my-vr-chart`.
