## Summary

**Banks**

- **City List import** — Import banks from City List screenshots with multi-image OCR, review/validation, and deposit capacity checks.
- **Smarter deposit slips** — Dedupe noisy OCR, link slips to commanders, persist outcomes, and upload incremental history videos (chunked for long clips) with a latest-slip hero.
- **Deposit dashboard** — Filterable fintech-style deposit stats and falloff projections for drop planning.

**My Kills, THP & Discord**

- **My Kills + KillScore** — Track kills on HQ, sync with Ashed, and upload alliance kills leaderboards to KillScore.
- **Discord charts & commands** — VR/THP progress charts as PNG slash replies; `/kills` / `/thp` growth replies; channel-visible VR reports and status queries.
- **THP OCR locales** — Power Details OCR hardened across DE, pt-BR, KO, and es-MX with a partial-parse breakdown form.

**Trains & connect**

- **Trains Simple Mode** — Guided conductor flow for new officers (default on), plus TPIF defaults, settings UX, and last-member pool guarantee.
- **Link a device** — Dedicated multi-device linking page with Ashed rebind warning, copyable pairing URL, sticky Connect CTA, and Ashed status in the shell header.

**Video & ops**

- **VS Daily/Weekly** — Choose Daily or Weekly when submitting VS Performance; weekly dates anchor to server Sunday.
- **Clearer review** — Live progress on the waiting page, better match cues, and Connect Ashed CTAs when rematch/submit need credentials.
- **External video worker** — Phase 2b worker host on main; admin jobs group primary/shadow passes.

**Also in this release**

- War Leader Support (Engineer–WL pairing), native Data Management with batch RBAC, battle-plan stronghold capture reminders, and fixed welcome/claim share links that auto-redeem instead of 404ing.

## Breaking changes

- None for operators. Trains Simple Mode defaults **on** per user (toggle off in trains user settings to restore the dense quick-actions grid).
