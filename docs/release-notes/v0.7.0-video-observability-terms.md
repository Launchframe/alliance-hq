## Working notes

- Public Terms of Service page at `/terms` (en-US and pt-BR), MIT license, footer link from connect flow.
- Video upload: clearer messaging on the 4 MB file size limit.
- Video review: save works when no Ashed events exist yet — auto-provision AllianceExercise / ZombieSiegeEvent on submit.
- Video review: score warnings and sorted member dropdown on the review page.
- Block 0-score validation for Zombie Siege and VS event results.
- Score reconciliation: resolve duplicates when OCR returns different score units for the same row.
- Admin video jobs list and per-job detail page with Frames, Parse results, and Timing tabs.
- Per-frame OCR error handling; pipeline stores upload/extract timing, OCR counts, and raw OCR JSON on `video_frames`.
- Job-level `timings_json` and total file size on `video_jobs` (migration `0013_video_diagnostics`).

## Summary

- Public **Terms of Service** page at `/terms` (English and Portuguese), plus an MIT license in the repo.
- **Video review** saves even when Ashed has no matching event yet — HQ auto-creates the event on submit.
- **Score warnings** on the review screen; invalid zero scores are blocked for VS and Zombie Siege results.
- **Duplicate score rows** from mixed OCR units are reconciled instead of double-counting.
- **Platform admins** get a video jobs list and per-job diagnostics: frame thumbnails, parse output, and timing breakdown.
- Upload form shows a clearer message when a file exceeds the **4 MB** limit.
